# Digital Marketing Project 2 – SEO Basics & Keyword Research

## Overview
This repository contains my submission for **Project 2: SEO Basics & Keyword Research** as part of my Digital Marketing internship/training.

The objective of this project is to understand the fundamentals of Search Engine Optimization (SEO) by performing keyword research, identifying search intent, and selecting relevant keywords for content planning.

## Project Objectives
- Understand the basics of SEO.
- Perform keyword research for a website or blog.
- Identify user search intent.
- Recommend relevant keywords for content creation.
- Build a foundation for data-driven digital marketing strategies.

## Topics Covered
- SEO Fundamentals
- Keyword Research
- Search Intent Analysis
- Primary & Secondary Keywords
- Long-tail Keywords
- Content Keyword Planning

## Skills Demonstrated
- Search Engine Optimization (SEO)
- Keyword Research
- Search Intent Analysis
- Content Planning
- Digital Marketing Fundamentals
